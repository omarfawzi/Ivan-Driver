# iVan Driver

## How to use?

1. Download or clone this repo.
```
git clone https://github.com/omarfawzi/Ivan-Driver.git
```
2. Install dependencies.

```js
npm install
// or
yarn install
```

3. Run project

```js
 npm start
```

4. Generate Apk

````
- cd android && ./gradlew assembleRelease
- react-native run-android --variant=release
````
